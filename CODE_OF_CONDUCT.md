# 🤝 Code de Conduite du Contributeur

## Notre engagement

Dans l'intérêt de favoriser un environnement ouvert et accueillant, nous, contributeurs et mainteneurs de **Homebrew-macOS-Toolchain-Optimizer**, nous engageons à faire de la participation à ce projet une expérience exempte de harcèlement pour tout le monde, quel que soit :

- Le niveau d'expérience technique
- Le genre, l'identité de genre ou l'expression de genre
- L'orientation sexuelle
- Le handicap
- L'apparence personnelle, la taille ou le poids
- L'origine ethnique, la nationalité ou l'âge
- La religion ou les convictions personnelles
- Tout autre critère discriminatoire

Nous valorisons les contributions de toutes les personnes, qu'elles soient techniques, documentaires, ou simplement sous forme de retours constructifs.

## Nos standards

### Comportements encouragés ✅

- **Respect et bienveillance** : Traitez les autres avec courtoisie, même en cas de désaccord technique.
- **Collaboration constructive** : Proposez des alternatives plutôt que de simplement critiquer.
- **Accessibilité** : Utilisez un langage clair, évitez le jargon excessif, et aidez les nouveaux contributeurs.
- **Reconnaissance** : Citez les sources, créditez les idées et remerciez les contributions.
- **Focus sur le projet** : Gardez les discussions centrées sur l'amélioration de la documentation et des outils macOS/Homebrew.

### Comportements inacceptables ❌

- **Harcèlement** : Commentaires désobligeants, attaques personnelles, intimidation ou comportements sexualisés.
- **Trolling** : Provocations intentionnelles, spam, ou détournement des discussions.
- **Discrimination** : Propos ou actions basés sur des stéréotypes ou préjugés.
- **Publication non consensuelle** : Divulguer des informations privées d'autrui sans consentement explicite.
- **Comportement non professionnel** : Insultes, langage agressif, ou refus de collaborer de bonne foi.

## Responsabilités des mainteneurs

Les mainteneurs du projet s'engagent à :

1. **Clarifier les standards** : Ce document définit les attentes comportementales.
2. **Agir de manière équitable** : Traiter toutes les préoccupations avec impartialité et confidentialité.
3. **Prendre des mesures correctives** : Supprimer, modifier ou rejeter les contributions non conformes.
4. **Communiquer les décisions** : Expliquer les raisons des actions disciplinaires lorsque cela est approprié.
5. **Protéger la communauté** : Restreindre temporairement ou définitivement l'accès aux personnes en violation répétée.

## Champ d'application

Ce code de conduite s'applique :

- ✅ Dans tous les espaces du projet : issues, PR, discussions GitHub, README, documentation
- ✅ Lors d'interactions publiques représentant le projet (réseaux sociaux, conférences, forums)
- ✅ Dans les communications privées liées au projet si elles impactent la communauté

Il ne s'applique pas aux interactions purement personnelles et extérieures au projet.

## Signaler un incident

Si vous êtes témoin ou victime d'un comportement inacceptable :

1. **Contactez directement** : [valorisa](https://github.com/valorisa) via message privé GitHub
2. **Décrivez l'incident** : Incluez dates, contextes, captures d'écran ou liens si possible
3. **Confidentialité garantie** : Votre signalement sera traité avec discrétion

> 🔒 **Note** : Si vous ne vous sentez pas à l'aise de contacter le mainteneur principal, vous pouvez également signaler l'incident via le formulaire de signalement de GitHub : [github.com/contact](https://github.com/contact)

## Conséquences en cas de non-respect

Les comportements inacceptables peuvent entraîner, selon la gravité et la récurrence :

| Niveau | Action | Exemple |
|--------|--------|---------|
| 🔹 Correction | Rappel privé des standards | Commentaire légèrement déplacé |
| 🔸 Avertissement | Notification formelle avec mise en garde | Comportement répété après rappel |
| 🔺 Suspension temporaire | Exclusion des discussions/PR pour une durée définie | Harcèlement modéré ou refus de collaborer |
| 🔴 Bannissement permanent | Exclusion définitive du projet et de ses espaces | Harcèlement grave, discrimination, menaces |

Les décisions sont prises par les mainteneurs, avec possibilité de recours motivé.
